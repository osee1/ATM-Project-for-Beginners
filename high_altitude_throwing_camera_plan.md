# 40层小区高空抛物监控采购和自安装方案

生成日期：2026-04-23  
预算目标：1000-4000元  
核心需求：只监控楼的一面、室外稳定使用、能记录/检索高空抛物轨迹、手机远程查看、尽量自己安装  

## 先给结论

最推荐买：**方案二，TP-LINK 800万双目一体机方案**。如果这台报价太高，再买**方案三，大华单800万变焦方案**。

原因很简单：你现在明确只监控楼的一面，不需要覆盖多面楼体，所以优先减少摄像机数量和施工量。40层楼大约有100-120米高度，普通家用摄像机不够用，但一台高空抛物专用800万双目/变焦摄像机已经可以作为主方案。双400万分段拍仍然更稳，但它属于“增强取证方案”，不是最低成本首选。

自己安装能不能做：

| 场景 | 能否自己做 | 难度 |
|---|---:|---:|
| 摄像机装在一楼外墙、岗亭、围墙、车棚、立杆等低处，只需要打孔和拉网线 | 可以自己做 | 3/5 |
| 摄像机装在室外低处，但需要走100米左右室外网线、穿管、防水 | 可以自己做，但要耐心 | 3.5/5 |
| 需要爬楼外墙、从窗外伸手固定、在高处打孔、楼顶边缘作业 | 不建议自己做 | 5/5 |
| 只做NVR、硬盘、App绑定、参数设置 | 很适合自己做 | 2/5 |

重要提醒：**高空抛物监控不是把摄像头装在40楼往下看，而是通常装在楼前地面/低层位置，向上拍整栋楼外立面。** 如果你的安装点必须高空作业，别自己上，找物业或弱电师傅做固定和走线，你自己负责选型、调试、手机绑定即可。

### 为什么这样配

TP-LINK官方高空抛物方案里，对26-45层高层楼栋给过类似思路：可以用高分辨率单机，也可以用两台400万分别覆盖上下区域。你只监控一面楼时，双机不是必须，但它的好处是低层和高层能分别拍清楚：

- 4mm焦距：覆盖低层和中低层，画面宽，便于看整体；
- 12mm焦距：盯中高层和高层，物体更大，轨迹更容易判断；
- PoE NVR：一根网线同时供电和传输，比Wi-Fi稳定；
- 4TB硬盘：比SD卡可靠，连续录像和事件录像都能保留。

### 优点

- 覆盖40层比单摄像头稳，尤其是高层细节更有把握。
- 同品牌摄像机和NVR，兼容性好，手机远程、录像检索、事件报警更省心。
- 两台分段拍摄，高层物体不会被压得太小。
- PoE有线供电，室外稳定性明显好于Wi-Fi。
- 后期可再加1-2台摄像机，因为NVR是4路。

### 缺点

- 施工比单摄像头多一根网线、多一个安装点；你只监控一面楼时，这个成本不一定必须花。
- 必须确认买到的是高空抛物专用型号，不能买成普通400万枪机。
- 要自己调角度和检测区域，第一次调试可能需要半天。
- 如果楼前树木、雨棚、广告牌遮挡严重，效果会下降。

- 需要确认所购具体型号支持“高空抛物轨迹/事件检索”，不要只看标题里的AI字样。
- 如果后续加第二台，总价可能接近或超过方案一。

- 只监控一面楼，且你能接受后期如果效果不够再加第二台。

如果你想一次到位，仍然优先方案二；如果你想省钱先落地，方案三更合适。

## 自己安装教程

### 第一步：现场测量和选安装点

1. 站在楼前，找一个能完整看到楼体外立面的地方，例如一楼外墙、门岗、围墙、车棚、立杆。
2. 尽量让摄像机离楼体10-25米，不要太近。太近会仰角过大，高层画面变形严重。
3. 避开树、晾衣杆、空调外机密集遮挡、路灯直射、广告牌。
4. 不要隔着玻璃拍，夜间会反光，红外灯也会打回玻璃。
5. 先用手机在预计安装点拍一张照片，确认整栋楼外立面都在画面内。

只监控一面楼时，优先把一台摄像机装在这面楼的正前方或斜前方，尽量让楼体外立面完整进入画面。单摄像机方案先让画面覆盖1-40层，再检查高层是否足够清楚；如果高层太小，再考虑加第二台分段拍。

方案一双摄像机的角度建议：

| 摄像机 | 焦距 | 主要负责 |
|---|---:|---|
| 低中层摄像机 | 4mm或6mm | 1-20层左右 |
| 中高层摄像机 | 12mm，现场距离远可考虑16mm | 15-40层左右 |

两台画面要有重叠区域，不要刚好断开。比如低层机看到1-22层，高层机看到16-40层，中间有几层重叠，后期更好判断轨迹。

### 第二步：先在室内把NVR和硬盘装好

1. 断电状态下打开NVR外壳。
2. 把4TB监控硬盘固定到NVR硬盘位。
3. 接好SATA数据线和电源线。
4. 合上外壳，NVR接显示器或电视HDMI。
5. NVR接路由器网口。
6. 开机初始化，设置强密码。
7. 进入硬盘管理，格式化硬盘。
8. 设置系统时间，开启NTP网络校时。

### 第三步：拉网线和防水

PoE方案只有一根网线到摄像机，这根线同时负责供电和视频传输。

1. 从NVR的PoE口拉室外六类网线到摄像机位置。
2. 室外部分尽量穿PVC线管或波纹管。
3. 摄像机接头必须放进防水接线盒。
4. 网线进入防水盒前做一个向下的弯，叫滴水弯，避免雨水顺线流进盒子。
5. 水晶头压好后用测线仪测试8芯是否全通。
6. 室外长线建议加PoE防雷器，尤其是立杆、楼顶、空旷位置。

不要这样做：

- 不要把水晶头直接裸露在室外；
- 不要用室内扁网线走室外；
- 不要用Wi-Fi代替PoE；
- 不要用普通插排放在室外给摄像机供电。

### 第四步：固定摄像机

1. 先把支架固定在墙面、立杆或金属架上。
2. 打孔用膨胀螺丝，固定后用手晃一下，不能松。
3. 摄像机先不要完全拧死，方便调角度。
4. 接上网线，看NVR实时画面。
5. 一个人在NVR或手机上看画面，另一个人微调摄像机角度。
6. 调好后再锁紧螺丝。

如果要站梯子安装：

- 两个人一起做；
- 梯子要有人扶；
- 不要在雨天、大风天安装；
- 超过2米且没有稳定平台，不建议自己做。

### 第五步：把摄像机加入NVR

同品牌PoE NVR通常会自动发现摄像机。

1. 摄像机网线接NVR的PoE口。
2. 在NVR里打开“通道管理”。
3. 等待自动添加。
4. 如果没有自动添加，手动搜索设备。
5. 设定摄像机密码，建议和NVR强密码一致或单独记录。
6. 确认每路都能实时预览。

### 第六步：设置录像

建议这样设置：

| 参数 | 建议值 | 为什么 |
|---|---|---|
| 录像方式 | 24小时连续录像 + 高空抛物事件标记 | 避免漏掉事件前后过程 |
| 预录时间 | 5-10秒 | 能看到物体出现前画面 |
| 延后录像 | 15-30秒 | 能看到落地后情况 |
| 编码 | H.265或H.265+ | 节省硬盘 |
| 分辨率 | 摄像机最高分辨率 | 取证优先 |
| 帧率 | 20-25fps | 抛物运动更连续 |
| 码率 | 400万建议4096-8192Kbps，800万建议8192-12288Kbps | 码率太低会糊 |
| 时间水印 | 开启 | 取证需要准确时间 |

4TB硬盘能保存多久取决于路数、码率和是否全天录像。粗略估算：

| 摄像机数量 | 码率设置 | 4TB大概保存 |
|---:|---:|---:|
| 1路400万 | 4096Kbps | 约80-90天 |
| 2路400万 | 每路4096Kbps | 约40-45天 |
| 2路400万 | 每路8192Kbps | 约20-25天 |
| 1路800万 | 8192Kbps | 约40-45天 |

取证场景我更建议清晰度优先，2路400万可以接受20-45天保存期。

### 第七步：设置高空抛物检测

不同品牌菜单名称不完全一样，但思路一致。

1. 打开智能事件或AI事件。
2. 选择高空抛物检测。
3. 在画面中只框楼体外立面，不要框路面、树、天空、马路。
4. 对住户室内窗口做隐私遮蔽，尽量只拍外立面、阳台外沿、空调位。
5. 开启轨迹绘制或轨迹记录。
6. 开启事件录像、事件截图、手机推送。
7. 灵敏度先设中等偏高，例如60-75。
8. 夜间误报多就略降灵敏度，白天漏报就略升。
9. 把树枝摇晃、路灯、旗帜、雨棚边缘设为屏蔽区域。

重点：检测区域不要贪大。只框你真正要监控的楼面，效果反而更好。

### 第八步：手机远程查看

TP-LINK大致流程：

1. 手机安装“TP-LINK安防”App。
2. NVR接入家里路由器。
3. NVR菜单里开启云服务或远程访问。
4. 用App扫码绑定NVR。
5. 打开实时预览、录像回放、事件消息。

海康大致用“海康互联”或“萤石云视频”。  
大华大致用“DMSS”或“乐橙”。

手机远程安全设置：

- NVR和摄像机都要强密码；
- 不要用123456、admin123这类密码；
- App账号开手机验证码；
- 不要把监控分享给无关人员；
- 不要公开视频画面。

### 第九步：测试

不要真的从高楼往下扔东西测试。可以这样安全测试：

1. 白天看实时画面，确认1-40层都在覆盖范围内。
2. 晚上再看一次，确认路灯、反光、红外没有把画面打白。
3. 用一根绳子在楼前安全区域吊一个小物体，从低处缓慢移动，测试轨迹显示和事件录像。
4. 或在封闭安全区域，从2-3米高度放下软物体测试，不要影响行人和车辆。
5. 在手机App里回放，确认能按事件检索到录像。
6. 导出一段录像，确认时间水印、轨迹、画面都在。

## 关键参数要看什么，以及为什么

| 参数 | 建议 | 为什么重要 |
|---|---|---|
| 是否高空抛物专用 | 必须明确支持高空抛物检测、轨迹记录、事件检索 | 普通移动侦测只能提示画面动了，不能可靠标记抛物轨迹 |
| 分辨率 | 至少400万，优先400万双机或800万单机 | 高层小物体很小，分辨率不够会糊成一个点 |
| 焦距 | 双机建议4mm/6mm + 12mm/16mm；单机建议变焦 | 焦距决定看多宽和看多远，40层不能只看广角 |
| 帧率 | 20-25fps | 抛物速度快，帧率低会断续，轨迹不连贯 |
| 快门和夜视 | 支持宽动态、低照度、补光，夜间能看清 | 夜里物体运动容易拖影，路灯和窗口会造成反差 |
| 防护等级 | 室外至少IP66，优先IP67 | 长期风吹雨淋，防水差会坏 |
| 供电方式 | PoE有线优先 | 一根网线供电+传输，稳定，维护简单 |
| NVR兼容性 | 摄像机和NVR尽量同品牌 | 智能事件和轨迹功能跨品牌可能不能完整显示 |
| 硬盘 | 监控专用盘，不用普通移动硬盘 | 监控盘适合7x24小时写入 |
| 隐私遮蔽 | 必须支持 | 小区场景要避免拍进住户室内 |
| 手机App | 必须支持实时预览、回放、事件推送 | 你需要远程查看和及时发现事件 |

## 最容易踩坑的地方

1. 买成普通家用摄像头。它能看画面，但高空抛物取证能力弱。
2. 买成普通枪机。标题里有AI不等于支持高空抛物轨迹。
3. 只买一台超广角。看得很全，但高层小物体可能太小。
4. 用Wi-Fi。室外距离远、墙体多，稳定性不如PoE。
5. 只用SD卡。卡坏、容量小、回放检索都不如NVR硬盘。
6. 镜头焦距买错。40层千万别只买默认2.8mm/4mm就结束。
7. 夜间不测试。白天清楚不代表夜里能看清运动物体。
8. 没做防水盒。很多室外故障不是摄像机坏，是网线接头进水。
9. 没做隐私遮蔽。拍到住户室内会引发隐私纠纷。
10. 摄像机和NVR跨品牌混用。实时画面可能能看，但智能事件和轨迹不一定能用。

## 合规和隐私注意

小区公共区域安装监控，最好先和物业、业委会或相关业主沟通。安装时注意：

- 只拍楼体外立面和公共区域，避免拍进住户室内；
- 使用隐私遮蔽把窗户内部区域挡住；
- 不要把监控画面公开传播；
- 明确保存周期和查看权限；
- 如果是公共区域固定安装，建议由物业或业委会留档。

这个不是小题大做。高空抛物监控本来是为了公共安全，但如果角度和权限处理不好，也容易变成隐私争议。
